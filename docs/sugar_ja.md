# Sugar.pm リファレンス

## 概要

Sugar.pm は chabo-dsl の糖衣構文を処理するモジュールです。

|関数|説明|
|----|----|

# convert

## 機能

日本語スクリプトをchabo-dsl構文に変換する

## 構文

```perl
convert($script)
```

## 引数

変換元文字列

## 戻り値

変換後の文字列を返す

# 自然言語変換

## 全角半角変換

* 全角ASCII → 半角ASCII
* 全角space → 半角SPACE
* 、 → 半角SPACE
* 。 → ;

## 条件文変換

もし～なら～以外は～
```
        もし(.+?)(?:なら|の場合|の時|時)(?:は)*(.+?)以外(?:(?:は|で))?
```
### 比較演算子
```
my %jp_op = (
    '未満'     => '<',
    '以下'     => '<=',
    '超える'   => '>',
    '以上'     => '>=',
    '等しい'   => '==',
    '異なる'   => '!=',
    '以外'     => '!=',
    'じゃない' => '!=',
    'でない'   => '!=',
);
```
### 論理演算子
```
my %jp_logical = (
    'で'     => '&&',
    'かつ'   => '&&',
    'か'     => '||',
    'または' => '||',
);
```
### 変換例

```
もし信号が青か緑なら進んでも良い以外で信号が黄なら注意以外は止まれ
　　　↓↓↓
信号 == 青 || 信号 == 緑 ? 進んでも良い 
                         : 信号 == 黄 ? 注意
                                      : 止まれ
```
## ユーザー定義関数変換

(引数)で(関数名)には(処理)
```
        ・(.+?)で(.+?)には(.+?);
```

複数の引数は`で`で区切る
```
単価と数量で金額を求めるには単価＊数量
　　　↓↓↓
金額を求める(単価,数量) = 単価*数量
```
## 関数を抜ける(return)

金額を返す → return(金額)

## 関数を繰り返す

最初に戻る → continue()

## 範囲演算子

10から30 → 10..20

## 対応している変換

Sugar.pm は現在、次の自然言語構文を chabo-dsl 構文へ変換します。

- 全角・半角文字の正規化
- 条件文
- 比較演算子
- 論理演算子
- ユーザー定義関数
- return 文
- continue 文
- 範囲演算子

今後のバージョンでは、対応する自然言語構文が追加される場合があります。

---

## 注意事項

- Sugar.pm は日本語スクリプトを機械的に変換します。
- 文法的に曖昧な日本語は期待どおりに変換できない場合があります。
- 変換後のコードは Ast.pm により実行されます。
