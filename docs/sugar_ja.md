# convert

## 機能

日本語スクリプトをchabo-dsl構文に変換する

## 構文

```perl
convert($script)
```

## 引数

変換元文字列

## 返り値

返還後の文字列を返す

# 自然言語変換

## 全角半角変換

* 全角ASCII → 半角ASCII
* 全角space → 半角SPACE
* 、 → 半角SPACE
* 。 → ;

## 条件文変換

もし～なら～以外は～
```
        もし(.+?)(?:なら|の場合|の時|時)(?:は)*(.+?)以外(?:(?:は|で))?
```
### 比較演算子
```
my %jp_op = (
    '未満'     => '<',
    '以下'     => '<=',
    '超える'   => '>',
    '以上'     => '>=',
    '等しい'   => '==',
    '異なる'   => '!=',
    '以外'     => '!=',
    'じゃない' => '!=',
    'でない'   => '!=',
);
```
### 論理演算子
```
my %jp_logical = (
    'で'     => '&&',
    'かつ'   => '&&',
    'か'     => '||',
    'または' => '||',
);
```
## ユーザー定義関数変換

(引数)で(関数名)には(処理)
```
        ・(.+?)で(.+?)には(.+?);
```

複数に引数は`で`で区切る
* 単価と数量で金額を求めるには単価＊数量

## 関数を抜ける(return)

金額を返す → return(金額)

## 関数を繰り返す

最初に戻る → continue()

## 範囲演算子

10から30 → 10..20
